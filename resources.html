Which Sensors Do I Have In My Smartphone? How Do They Work? Manisha Priyadarshini This article written by Manisha Priyadarshini from Fossbytes briefly describes different sensors that smartphones commonly have and how they are used in the real world. The sensors range from mundane sensors like GPS and Fingerprint sensors to more exotic sensors like Geiger Counters and Air Humidity sensors. This article would be useful for my class because it shows how some of the most common sensors on smartphones are used in the real world which can give me inspiration for my own projects that take advantage of sensors. Step-By-Step Guide To Implementing A Fingerprint Scanner In Your Android Application JetRuby Agency This article written by The JetRuby Agency on Jetruby.com is a step by step guide on how to implement a fingerprint scanner into an android application. It focuses on the FingerprintAPI and classes within this API like FIngerprintManager and FingerprintManagerCompat that are used for interfacing an android phone's fingerprint sensor with an application. For security, the use of Android KeyStore is shown to help keep fingerprint information secure. A KeyPairGenerator is created from KeyStore to implement an encryptor and decryptor that helps store fingerprint data. The article shows the code that is used to complete these tasks and explains in great detail the process of implementing this fingerprint sensor solution.
This article will be very useful as it explains both the concepts and code that I might need to know to implement fingerprint sensors in an application. Especially since I am considering using fingerprint sensors for upcoming mobile development projects. 		10

The Sensors That Make the iPhone so Cool Sam Costello This article written by Sam Costello on Lifewire.com briefly describes the different sensors that exist on iPhones and how they are used. While most of the sensors on here are common amongst all smartphones, the article does also talk about iPhone-specific sensor features like Face ID which can provide authentication for Apple services like Apple Pay.
